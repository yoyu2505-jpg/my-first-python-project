# ✈️ Trip.com 航班數據自動化抓取工具 (QA Automation Version)

這是一個基於 Python 與 Selenium 開發的自動化航班數據爬蟲工具。本專案專為 QA 自動化測試與數據收集設計，具備友善的命令列（CLI）互動介面，支援多步驟輸入、防呆驗證以及隨時回退（上一步）的功能。

自動化腳本會模擬真人瀏覽器操作，依據使用者輸入的條件搜尋 Trip.com 的來回航班，並將完整的去回程航班資訊（含航空公司、時間、機場、價格等）自動匯出為 CSV 檔案。

---

## ✨ 核心特色

* **互動式引導介面**：5 步驟互動輸入，支援輸入 `b` 回到上一步修正資料，或輸入 `q` 隨時退出。
* **智慧型日期選擇器**：使用高效的 JavaScript 注入技術，自動處理動態網頁日曆的滾動與點擊，克服跨月份選擇的痛點。
* **動態加載等待**：內建遮罩層（Loading Overlay）智慧偵測，自動等待網頁載入完成，大幅降低因網頁延遲導致的元素定位失敗率。
* **路徑防錯機制**：自動識別執行環境，無論是直接執行 `.py` 腳本或使用 PyInstaller 打包成 `.exe` 執行檔，產出的 CSV 檔案皆會正確儲存在程式同級目錄下。
* **防亂碼編碼**：CSV 檔案採用 `utf-8-sig` 編碼格式，確保在 Windows Excel 中開啟時中文不產生亂碼。

* **核心語言**：Python 3.x

## 📋 擷取數據欄位 (15 個欄位)

程式會將抓取到的每組來回機票組合，細分解析並儲存為以下結構：

| 去程資訊 | 回程資訊 | 價格資訊 |
| :--- | :--- | :--- |
| 去程航空公司、去程日期、去程耗時 | 回程航空公司、回程日期、回程耗時 | 總價格 (TWD) |
| 去程出發時間、去程出發機場 | 回程出發時間、回程出發機場 | |
| 去程抵達時間、去程抵達機場 | 回程抵達時間、回程抵達機場 | |
